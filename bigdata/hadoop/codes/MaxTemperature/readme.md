## 1. 使用方法
### 1-1. 单机模式
1. 确认环境
    1. 单机模式,jps里没有任何数据
    2. 我的实验环境中`hadoop/etc/hadoop`是单机模式的`hadoop-local`目录的软连接
2. 准备数据
3. 编译java程序,打包成jar,使用hadoop执行
4. 验证
